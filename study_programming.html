02. 텍스트 위계와 서식 속성

텍스트 태그는 단순히 글자를 보여주는 것이 아니라, 문서의 **'중요도'**를 결정합니다.

제목 태그 (Heading)

<h1> ~ <h6>숫자가 작을수록 중요한 제목이며, 검색 로봇은 h1을 보고 페이지 주제를 파악합니다.

본문 강조 태그

<p>Paragraph의 약자로, 하나의 문단을 만듭니다. (Block 요소)
<strong> / <b>굵은 글씨. strong은 '중요한 의미'를 담고, b는 '시각적'으로만 굵게 만듭니다.
<em> / <i>기울임꼴. em은 '강조된 의미', i는 '단순 기울임'입니다.
<mark>형광펜 효과(하이라이트)를 줍니다.

03. 링크(<a>)의 무궁무진한 속성

링크 태그는 웹사이트 이동뿐만 아니라 이메일, 전화, 파일 다운로드까지 담당합니다.

href="주소"연결할 목표 주소입니다.
href="mailto:이메일"클릭 시 사용자의 메일 앱이 열리며 주소가 자동 입력됩니다.
href="tel:전화번호"모바일에서 클릭 시 바로 전화를 겁니다.
target="_blank"새 탭에서 링크를 엽니다. (생략 시 현재 탭에서 이동)
download이동하지 않고 링크된 파일을 바로 저장합니다.

01. 시맨틱 레이아웃

의미가 있는 태그를 써야 구글이 우리 사이트를 좋아합니다.

<header>상단(로고, 메뉴)
<nav>주요 링크(메뉴바)
<main>가장 중요한 본문
<footer>하단 정보(저작권)
